--- layout: tool title: Name Generator description: "A tool to generate names using the haiku approach popularized by Heroku. I use this for creating names of ships in the Postal Marines series." ---

This name generator uses the "haiku" approach popularized by Heroku. Because I wanted to create terms using Esperanto, I included a translation. There are 98 adjectives and 103 nouns, creating now fewer than 17 permutations; or now fewer than 34 permutations when including the number range from 1000 to 9999.
